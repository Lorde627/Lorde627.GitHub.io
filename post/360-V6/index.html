百元最强路由器: 360 V6 软刷 QSDK OpenWrt 固件

2022-01-05 / 7 min read

随着新一代高通IPQ系列 WI-FI 6 路由器登场,风靡一时的 MTK MT762x 路由器也将踏入退休之路.
如今路由器生态的进一步发展以及开源开发者们的无私奉献,如今的无线路由器已不仅仅只是提供无线接入,其有了更多的可玩性.

360 V6 搭载了高通IPQ 6000 平台

  • 配备4颗Skyworks的信号放大器保证信号覆盖

QBittorrent

下载轻度用户考虑可以外接一个硬盘,使用固件自带的 QBittorrent 进行下载,通过SMB共享也能充当一个小下载机.不过由于 USB 3.0 会影响 2.4G频段信号,厂商把接口阉割成 USB 2.0,速度有些拉垮.
经过ACWIFI的测试,USB读取速度为33.5MB/s，写入速度为25MB/s.

如果你想了解更多360 V6 硬件信息,请访问 ACWIFI 拆解详情: https://www.acwifi.net/10309.html

2. 解锁 Telnet

使用网线连接电脑,输入 192.168.0.1 进入路由器后台,进入插件页面,选择安装 360_v6_jd_telnetd_ctrl_app_nda_signed.opk 插件,进而打开Telnet开关.

3. 刷写Uboot和大分区文件

下载 Putty 与 HFS.
Putty: https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html

HFS:https://www.nb-fk.com/usr/uploads/2020/11/958029150.zip

进入HFS
将 360v6-mibib.bin 与 uboot-qihoo-v6.bin 文件拖入左框.
点击右键复制两个文件的 URL.

进入 Putty:
Host Name 填写 192.168.0.1
Connection Type 选择 Other:Telnet.

输入用户名: root
密码: telnetdebug

输入代码挂载tmp:
cd /tmp

输入 wget https://你的IP/uboot-qihoo-v6.bin (刚刚复制的URL)
wget https://你的IP/360v6-mibib.bin

输入 cat /proc/mtd 查看分区
确认大分区在 1 分区,Uboot 在 13 和 14 分区.

确认无误后开始刷写:

mtd write /tmp/360v6-mibib.bin /dev/mtd1 (刷入大分区)
mtd write /tmp/uboot-qihoo-v6.bin /dev/mtd13 (UBoot 刷入第13分区)
mtd write /tmp/uboot-qihoo-v6.bin /dev/mtd14 (UBoot 刷入第14分区)

完成后请拔掉电源.
完成后请拔掉电源.
完成后请拔掉电源.

4. 进入Uboot不死刷写固件

再次插入电源并同时使用卡针或任何物件按住V6路由器Reset孔5秒直至红灯亮起.

电脑网络设置为手动填写,IP为192.168.1.10,网关为192.168.1.1.
浏览器输入 192.168.1.1 进入不死 Uboot.

选择群文件里的最新固件进行刷写即可,到这里刷机就成功啦!~🎉

无线问题 🆘

2022.01.19 目前新固件已解决无线速率的问题
不过唯一目前的不足是部分机器无线有点问题待处理,建议将无线5G功率调到18,信道选择在153-161,以获得正常的无线速率,期待后续固件开发者或许能修复这个问题.

若你在使用中遇到设备无法连接Wi-Fi的情况,请将加密模式改为 Mixed Mod.

路由器退休 👴

此前我使用是 斐讯K2P 路由器,在更换360 V6 路由器后完成了自己的光荣使命.
无论是 Pandavan 与 Lean开发的Pandora 固件无线都能满足跑满500M无线的速率.
超频1200Mhz后的K2P 使用Xray在实测中 YouTube 测速极限能达到5万左右,相当于5MB/s 的速度,实测YouTube,Netflix,Disney+ 4K内容都能流畅播放.

AdGuard Home 由于K2P的小内存,我将其放在猫盘NAS上通过DNS转发来使用,效果也不错.

其实斐讯 K2P 已经能完全满足我的需求,家里唯一支持 Wi-Fi6 的设备仅有一台iPhone,但无可奈何耐不住那颗爱折腾的心呐哈哈哈哈.

若你还在使用 斐讯K2P 路由器,在这里给大家推荐两个固件:
Pandavan: https://www.right.com.cn/forum/forum.php?mod=viewthread&tid=5553350&extra=page%3D8%26filter%3Dtypeid%26typeid%3D14

PandoraBox: 最新版在上述在的固件支持群的群文件里.
